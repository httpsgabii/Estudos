Questionário


QUESTÂO 1: Explique qual é a função de um servidor web em um ambiente de hospedagem e cite pelo menos um exemplo.

R: O servidor web é responsável por entregar os arquivos estáticos como HTML, CSS, imagens etc. para os usuários que acessam o site. Um exemplo é o Apache.
QUESTÂO 2: Qual a principal característica do NGINX e por que ele é muito utilizado em projetos de alto desempenho?

R: O NGINX é bem leve e eficiente, por isso é ótimo pra projetos de alto desempenho. Ele também funciona super bem como proxy reverso, o que ajuda bastante em sistemas grandes.
QUESTÂO 3: Diferencie os protocolos HTTP e HTTPS. Por que o HTTPS é recomendado para aplicações modernas?

R: HTTP é o protocolo padrão, mas sem segurança. Já o HTTPS tem criptografia com SSL/TLS, então protege os dados dos usuários. Hoje em dia, é essencial usar HTTPS pra garantir segurança e confiança.
QUESTÂO 4: Qual é a função das portas em uma rede de computadores e qual a importância da porta 443?

R: As portas são como canais de comunicação dentro de uma rede. A 443 é usada pro HTTPS, então ela garante que a comunicação seja segura.
QUESTÂO 5: Defina o conceito de endereço IP e explique sua importância em redes de computadores.

R: IP é o número que identifica um dispositivo na rede. Sem ele, os dados não saberiam pra onde ir. É como o “endereço” de uma casa, só que na internet.
QUESTÂO 6: Explique o que é hospedagem VPS e em quais situações ela é mais indicada.

R: A VPS é um servidor virtual mais potente, com mais controle. Eu usaria em projetos que precisam de mais desempenho ou personalização, como uma aplicação que cresce com o tempo.
QUESTÂO 7: Descreva as vantagens do servidor web Caddy em relação a outros servidores tradicionais.

R: O Caddy é super moderno, tem HTTPS automático e a configuração é simples. bom pra quem quer segurança e agilidade sem complicação.
QUESTÂO 8: Qual servidor de aplicação é mais indicado para aplicações desenvolvidas na plataforma .NET (C#)? Explique o motivo.

R: ASP.NET Core, sem dúvida. É a plataforma moderna da Microsoft, pensada pra APIs e aplicações web com desempenho e integração nativos com .NET.
QUESTÂO 9: Descreva o funcionamento do modelo Cliente-Servidor no contexto de sistemas web.

R: O cliente (normalmente um navegador) faz uma requisição e o servidor responde com o que foi pedido. É a base de como a web funciona.
QUESTÂO 10: Explique o que é o Apache Tomcat e para qual tipo de aplicação ele é mais indicado.

R: O Tomcat é um servidor de aplicação feito pra Java. Ele executa servlets e JSPs, sendo ótimo pra aplicações web baseadas nessa linguagem.
QUESTÂO 11: Diferencie, com suas palavras, um servidor web de um servidor de aplicação, destacando suas funções principais.

R: O servidor web só entrega arquivos prontos (estáticos). O de aplicação executa o código, interage com o banco de dados, gera conteúdo dinâmico. Um complementa o outro.
QUESTÂO 12: Defina o conceito de domínio na internet e sua relação com os endereços IP.

R: Domínio é o nome que a gente digita (tipo google.com). Ele está ligado a um IP, mas facilita a nossa vida porque ninguém vai ficar decorando números.
QUESTÂO 13: Quais as principais vantagens da hospedagem em nuvem em comparação com a hospedagem local?

R: A nuvem é escalável, distribuída e tem alta disponibilidade. Já a local depende de infraestrutura física, o que é mais limitado e arriscado.
QUESTÂO 14: Cite dois exemplos de servidores de aplicação e as linguagens/plataformas que eles suportam.

R: Node.js (JavaScript) – muito usado pra APIs e apps web. Django com Gunicorn (Python) – ótimo pra aplicações Python completas.
QUESTÂO 15: Por que é essencial a utilização de HTTPS em sites e sistemas atualmente? Quais riscos o HTTP apresenta?

R: O HTTPS protege os dados dos usuários. Sem ele (com HTTP), os dados podem ser interceptados facilmente, o que é um risco enorme pra qualquer sistema.
QUESTÂO 16: Dê um exemplo de situação onde o uso de hospedagem VPS seria mais vantajoso do que uma hospedagem compartilhada.

R: Um sistema de agendamento de consultas com muitos acessos simultâneos. A VPS garante mais desempenho e controle que a compartilhada.
QUESTÂO 17: Diferencie os conceitos de IP e porta em uma rede, explicando suas funções.

R: O IP identifica o dispositivo, a porta identifica qual serviço está sendo acessado nesse IP. É como saber a casa e o cômodo certo
QUESTÂO 18: Qual a importância de um servidor web conseguir lidar com conexões via HTTPS?

R: Essencial. Um servidor que não lida com HTTPS não é seguro e pode comprometer a confiança dos usuários e os dados que trafegam por ali.
QUESTÂO 19: Cite dois exemplos de servidores web e explique suas principais características.

R: Apache: popular, código aberto, super usado. LiteSpeed: compatível com Apache, mas bem mais rápido e otimizado.
QUESTÂO 20: Defina o que é uma hospedagem compartilhada e em quais casos ela costuma ser utilizada.

R: É quando vários sites dividem o mesmo servidor. Serve pra sites pequenos, blogs pessoais ou projetos que estão começando.
QUESTÂO 21: Explique a função de um servidor de aplicação e a diferença em relação a servidores que apenas servem arquivos estáticos.

R: O servidor de aplicação executa código, acessa banco de dados, gera respostas dinâmicas. O estático só entrega arquivos como HTML, imagens, etc.
QUESTÂO 22: Cite um servidor de aplicação usado para projetos em Python e descreva como ele funciona.

R: Django com Gunicorn ou uWSGI. Eles executam a lógica do app Python e lidam com requisições dinâmicas.
QUESTÂO 23: O que é o WildFly (ex-JBoss) e para qual tecnologia ele é indicado?

R: É um servidor de aplicação Java EE da Red Hat, voltado pra empresas. É bem completo e usado em ambientes corporativos.
QUESTÂO 24: Qual a função da porta 21 em redes de computadores e para qual serviço ela é utilizada?

R: É usada pro serviço de FTP – serve pra transferir arquivos entre o cliente e o servidor.
QUESTÂO 25:Explique por que o servidor LiteSpeed é compatível com o Apache e quais as vantagens dessa compatibilidade.

R: Ele entende os arquivos do Apache (.htaccess, por exemplo), o que facilita migrar de um pro outro. E o desempenho do LiteSpeed é bem melhor.
QUESTÂO 26: O que é o Microsoft IIS e em quais situações ele costuma ser adotado?

R: Servidor web da Microsoft, bem integrado ao Windows Server. É muito usado em empresas que usam .NET e ambientes corporativos com Windows.
QUESTÂO 27: Explique para que serve o Node.js e em quais tipos de aplicações ele é mais utilizado.

R: Node.js executa JavaScript no servidor. É ótimo pra aplicações em tempo real, como chats, APIs e sistemas que precisam de respostas rápidas.
QUESTÂO 28: Descreva a função do servidor GlassFish e para qual linguagem ele é destinado.

R: GlassFish é um servidor de aplicação completo pra Java EE. Perfeito pra apps corporativas em Java.
QUESTÂO 29: O que são Puma e Unicorn e em quais projetos eles são normalmente utilizados?

R: São servidores de aplicação usados com Ruby on Rails. São indicados pra lidar com alta concorrência e deixar o sistema ágil.
QUESTÂO 30: Qual a importância da escolha adequada entre hospedagem local, compartilhada, VPS e nuvem no planejamento de um projeto web?

R: Faz total diferença no desempenho e no sucesso do projeto. Um blog simples pode ir pra compartilhada, mas um sistema robusto e com picos de acesso vai muito melhor na nuvem ou VPS.
